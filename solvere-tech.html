Solvere Tech – Sistema de Delivery Personalizado por Assinatura

← Voltar ao portfólio

Descrição do Projeto

O Solvere Tech é um aplicativo de delivery desenvolvido para que restaurantes, bares e lanchonetes possam operar suas próprias plataformas de pedidos online, de forma personalizada e independente, por meio de um modelo de assinatura. Cada estabelecimento pode contratar o serviço, customizar o nome do app conforme sua marca, cadastrar seus próprios produtos e gerenciar todo o processo de pagamento pelo aplicativo. Além de permitir o cadastro e a personalização do cardápio, o sistema coleta feedbacks de clientes em relação aos pedidos realizados. Esses comentários são usados para identificar pontos de melhoria e potencializar os aspectos mais elogiados, ajudando o dono do estabelecimento a evoluir continuamente o serviço.

Principais objetivos

Permitir que cada restaurante tenha seu próprio app de delivery personalizado.

Integrar pagamentos diretamente pelo aplicativo (dinheiro, cartão ou pix).

Coletar feedbacks dos clientes para que os donos possam melhorar o atendimento.

Garantir que os empreendedores tenham autonomia para cadastrar produtos e personalizar o cardápio

Funcionalidades Implementadas

Tela de Home: login ou criação de conta para novos usuários.

Tela de Cadastro: novos usuários informam nome, telefone, e-mail e senha.

Tela de Cardápio: visualização dos produtos, adição ao carrinho, informações do restaurante e acesso ao perfil.

Tela de Pagamento: seleção de forma de pagamento e exibição de QR Code para pagamentos via pix.

Cart Context: gerenciamento do carrinho com métodos para adicionar, listar, atualizar e excluir produtos.

Integração com banco de dados

Inserção de novos usuários (INSERT)

Consulta de dados de clientes (SELECT)

Atualização dos dados cadastrados (UPDATE)

Exclusão de registros (DELETE)

Tratamento de exceções com feedback ao usuário usando blocos try-catch

Melhorias previstas

Permitir que os donos editem diretamente os itens disponíveis para venda

Implementar histórico de pedidos

Adicionar práticas de segurança, como autenticação de usuários e prevenção de SQL Injection

Tecnologias Utilizadas

  • Linguagem de programação: JavaScript
  • Framework: React Native
  • Banco de dados: PostgreSQL, acessado por meio de uma API de backend
  • Arquitetura: Programação orientada a objetos para modularidade e organização do código
